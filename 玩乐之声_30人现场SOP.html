01
PHASE 01
演出前 7 天 · 确认与准备
演出日期倒计时 7 天内完成
责任人：统筹 PM
确认演出乐队与曲目清单
与乐队确认演出曲目（至少4首）、演出时长（建议45-60分钟）、乐器编制、特殊设备需求
统筹发微信/电话确认
发送观众邀请与票务
确认30个名额分配：公开售票席 + 嘉宾/甲方邀约席（建议至少5席留给目标甲方）。通过大麦/秀动或私域社群发起，截止时间为演出前48小时
内容运营30人上限严格控制
发布演出预热内容
抖音/小红书/视频号发布预热短视频或图文，包含：乐队介绍、演出时间地点、购票/报名方式。要求乐队同步在自己账号转发
内容运营带话题标签
确认拍摄方案与机位设计
视频导演根据乐队编制确定机位数量（建议3-5机位）：固定主机位 + 手持移动机位 + 特写机位。确认录音与拍摄的协调方案，避免设备冲突
视频导演
设备检查与备品准备
制作总监检查：录音设备、调音台、话筒（含备用）、耳机、线材、DAW系统。视频导演检查：摄像机电池/存储卡、三脚架/稳定器、监视器。发现损坏立即处理
制作总监 + 视频导演备用电池必须充满
确认灯光方案
灯光师根据本场演出风格（安静内省 / 活跃律动）调整灯光预设方案，确认灯光不影响摄影机曝光（避免过曝或频闪问题）
灯光师
02
PHASE 02
演出前 3 天 · 确认与跟进
演出日期倒计时 3 天
责任人：统筹 PM
观众名单确认并建群
整理最终入场名单（30人），建立本场观众微信群。群内提前告知：入场时间、地址、停车、着装建议（可选）、不可拍摄/录像的提醒（如适用）
现场统筹超额报名需候补管理
乐队到场时间与技术需求二次确认
与乐队再次确认：到场时间（演出前3小时）、乐器自带清单、演出服/形象要求、是否需要返送监听、有无特殊Rider要求
统筹 PM
场地布置检查
确认观众席位布置（围绕乐手方案）、座椅数量（≤30）、动线是否顺畅、紧急通道是否畅通、空调/通风是否正常
现场统筹
邀约甲方最终确认
对本场邀约的5席目标甲方逐一电话/微信确认到场。若有临时无法到场的，立即补位替换，不要让席位空着
商务 / 统筹甲方席位必须坐满
物资采购
确认现场所需物资：饮水（观众人手一瓶）、现场物料（节目单/乐队介绍卡片）、名片、备用电池、延长线，统筹采购到位
现场统筹
03
PHASE 03
演出当天 · 场地准备
演出开始前 3 小时
责任人：制作总监 + 统筹
场地开门与基础检查
统筹最早到场，开门通风，检查空调温度（建议22-24°C）、照明、卫生间、消防设备状态
统筹 PM演出前 3h 到场
录音系统搭建与测试
制作总监完成：接线 → 调音台通道检查 → DAW工程文件建立（以演出日期命名）→ 录音电平测试 → 备份存储介质准备（至少2块硬盘）
制作总监必须双盘备份
拍摄设备架设
视频导演完成：各机位三脚架架设 → 对焦测试 → 白平衡统一设置 → 存储卡格式化（确认已备份上次素材）→ 监视器连接测试
视频导演存储卡清空前确认备份
灯光系统调试
灯光师完成所有灯位调试，与视频导演沟通确认灯光方向不产生摄影机眩光。预设演出各段灯光场景（开场/演奏中/安静段/高潮段）
灯光师
观众席布置
现场统筹完成：座椅按方案摆放（围绕乐手）→ 每个座位放置节目单/乐队介绍 → 入场引导标识 → 饮水准备 → 手机提示牌（请勿录像/请静音）
现场统筹
乐队接待到场
现场统筹迎接乐队，引导乐器进场路线（避免与观众动线冲突），安排乐队休息区，提供饮水，告知当天流程时间表
现场统筹
04
PHASE 04
调音彩排阶段
演出前 2 小时 · 严格保障
责任人：制作总监
调音时间是品质的核心保障，任何情况下不得压缩至1小时以下。若乐队迟到导致调音不足，须告知乐队对演出质量可能产生影响，并记录在案。
乐器摆位确认
制作总监与视频导演共同确认乐器摆位，确保：① 话筒/乐器在录音拾音最佳位置；② 所有乐手在摄影机视角内；③ 线材走线整洁不遮挡拍摄
制作总监 + 视频导演
分轨调音与电平设置
各乐器/人声分轨逐一测试 → 设置录音电平（保留6dB余量，避免削波）→ 监听返送调整（每位乐手确认自己的耳返/监听音量）
制作总监Peak不超过-6dBFS
试录一段检查
乐队演奏一首完整曲目，制作总监回放检查录音质量，视频导演检查各机位画面，灯光师微调。发现问题立即处理，不留到演出时
全体技术人员这是最后的检查机会
演出流程与曲目顺序确认
统筹与乐队确认：开场顺序 → 曲目顺序 → 有无曲目间互动/介绍 → 安可环节安排 → 结束时间预估 → 拍摄特殊需求（如走位/站位变化）
统筹 PM
开始录音（Pre-Roll）
演出前30分钟正式开始录音，打板记录（场次名称+日期+Take号），之后保持录音不停止直到演出完全结束。宁可多录，不要遗漏
制作总监录音一旦开始不要暂停
05
PHASE 05
演出进行中
开场至演出结束
责任人：统筹 PM 全局把控
观众入场引导（开场前30分钟）
现场统筹在门口核验名单（微信确认/纸质名单），引导观众就座。告知：请关闭手机铃声、不可使用闪光灯拍照、尊重演出氛围。甲方嘉宾优先引导至最佳位置
现场统筹迟到观众谢绝入场（演出开始后）
内容运营实时发布
内容运营在演出开始后15分钟内，发布"演出开始"实时内容（抖音/视频号直播或短视频），配合观众群实时互动。注意：不要在关键安静段发布会震动提示的内容
内容运营
录音持续监控
制作总监全程监控录音电平，关注：有无削波（红灯）、有无杂音/噪声、存储空间是否充足。发现问题立即静默处理，不要影响演出
制作总监
多机位拍摄执行
视频导演统筹各机位：主机位固定拍全景，移动机位寻找最佳情绪镜头，特写机位捕捉手部/表情细节。每首曲目至少保证一次全景 + 一次近景切换
视频导演
时间把控
统筹在后台关注演出时长，若超时10分钟，给乐队发送手势信号（非言语）提示时间。正常演出时长45-60分钟，安可1首，总时长不超过75分钟
统筹 PM
演出结束后录音延续
演出最后一首曲目结束后，录音继续保持至少5分钟（捕捉掌声、现场氛围），然后制作总监正式停止录音并打结束板记录
制作总监
06
PHASE 06
演出结束后 · 收尾与跟进
演出结束当晚 + 后续 72 小时
责任人：统筹 + 各岗位
素材即时备份（当晚完成）
制作总监：录音文件备份至两块硬盘，命名规则：「YYYYMMDD_乐队名_场次」。视频导演：所有存储卡素材备份至移动硬盘，原卡保留至剪辑完成后再格式化
制作总监 + 视频导演当晚必须完成，不可拖延
乐队送场与感谢
协助乐队搬运乐器，统筹当面感谢并告知后续制作时间节点（几天内出初剪/混音）。提醒乐队在自己社媒账号发布演出感谢内容并@我们账号
统筹 PM
现场收尾与场地恢复
所有设备归位，线材整理收纳，桌椅恢复，垃圾清理，关闭空调/照明，锁门。检查有无观众遗忘物品（如有联系归还）
现场统筹
演出当晚发布高光内容（24小时内）
内容运营从当晚素材中快速剪辑1-2条30-60秒高光视频，次日上午发布至抖音/视频号/小红书。趁观众热度最高时发布效果最好
内容运营 + 视频导演24小时内发布效果最佳
甲方嘉宾跟进（48小时内）
统筹或商务对本场邀约的目标甲方逐一发送：① 感谢参与的消息；② 1-2条演出精彩视频；③ 服务说明书（如对方有合作意向）。不要超过48小时，趁热打铁
商务 / 统筹48小时内必须跟进
完整版内容交付（7天内）
视频导演完成完整剪辑版（15-30分钟演出精华版），制作总监完成混音初稿。发送给乐队审核，确认后上传至B站/私有云存储，归档本场内容资产
制作总监 + 视频导演
本场复盘记录
统筹填写《演出复盘表》：观众反馈、技术问题记录、流程改进建议、甲方跟进状态、本场积分贡献初稿。复盘表归档至内部文件夹
统筹 PM
应急预案 · 常见突发情况处理
乐队成员临时缺席
立即联系备用乐手（维护至少3名可随时联系的备用乐手）。若无法补位，评估是否可以减员演出（如鼓手缺席是否可无鼓进行）。最差情况提前1小时告知观众并道歉，提供下次优先购票权。
设备故障（录音/摄像）
录音设备故障：切换备用录音设备（提前准备手机录音作为应急保底），告知乐队稍作等待。摄像设备故障：其余机位继续拍摄，故障机位重启处理，演出不因此暂停。
停电 / 断电
统筹立即联系场地物业处理。若短时间无法恢复，宣布短暂暂停，保持现场秩序，告知观众预计等待时间。超过20分钟无法恢复，启动演出取消应急方案（退票/改期）。
观众超员 / 无票入场
门口严格核验名单，30人满员后谢绝入场，无论关系远近。可安排候补观众在门口等待，如有人提前离场则补位。任何情况下不突破30人上限（安全及品质双重考虑）。
观众突发不适
立即引导至休息区，提供饮水，评估症状严重程度。若需要就医立即拨打120，统筹全程陪同，不要让伤者单独等待。演出继续，不要打断，但统筹不参与演出后续工作直到妥善处理。
乐队临时要求变更曲目
调音后要求变更：小幅调整（换1-2首）可接受，重新快速调试相关通道。大幅变更（超过一半曲目）须告知可能影响录音质量并记录在案，不承担相应质量责任。
